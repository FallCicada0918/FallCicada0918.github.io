Python 基础详解

1. Python 语法基础

Python 是一种易于学习和使用的编程语言。以下是 Python 的基本语法：

# 变量声明
x = 10
name = "FallCicada"

# 注释
# 这是单行注释
"""这是多行注释"""

# 缩进
if x > 5:
    print("x 大于 5")

2. 数据类型

Python 提供了多种内置数据类型：

  • 数字类型：int, float, complex
  • 字符串：str
  • 列表：list
  • 元组：tuple
  • 字典：dict
# 示例
num = 42
pi = 3.14
text = "Hello, Python!"
my_list = [1, 2, 3]
my_tuple = (4, 5, 6)
my_dict = {"key": "value"}

3. 控制结构

Python 支持条件语句和循环：

# 条件语句
if x > 0:
    print("x 是正数")
elif x == 0:
    print("x 是零")
else:
    print("x 是负数")

# 循环
for i in range(5):
    print(i)

while x > 0:
    print(x)
    x -= 1

4. 函数

函数是代码复用的重要工具：

# 函数定义
def greet(name):
    return f"Hello, {name}!"

# 调用函数
print(greet("FallCicada"))

5. 面向对象编程

Python 是一种面向对象的语言，支持类和对象：

# 定义类
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        return f"Hi, I am {self.name} and I am {self.age} years old."

# 创建对象
person = Person("FallCicada", 25)
print(person.greet())
