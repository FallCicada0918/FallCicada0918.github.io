Linux 常用指令详解

1. 文件和目录操作

以下是一些常用的文件和目录操作指令：

  • ls: 列出目录内容
  • cd: 切换目录
  • pwd: 显示当前工作目录
  • mkdir: 创建新目录
  • rm: 删除文件或目录
# 示例
ls -l
cd /home/user
pwd
mkdir new_folder
rm -r old_folder

2. 文件内容查看

以下指令用于查看文件内容：

  • cat: 显示文件内容
  • more: 分页显示文件内容
  • less: 分页显示文件内容，支持上下滚动
  • head: 显示文件的前几行
  • tail: 显示文件的后几行
# 示例
cat file.txt
more file.txt
less file.txt
head -n 10 file.txt
tail -n 10 file.txt

3. 权限管理

以下指令用于管理文件和目录的权限：

  • chmod: 修改文件权限
  • chown: 修改文件所有者
  • chgrp: 修改文件所属组
# 示例
chmod 755 script.sh
chown user:group file.txt
chgrp group_name file.txt

4. 系统管理

以下指令用于系统管理：

  • ps: 显示当前运行的进程
  • top: 实时显示系统资源使用情况
  • df: 显示磁盘使用情况
  • du: 显示目录或文件的大小
  • uptime: 显示系统运行时间
# 示例
ps aux
top
df -h
du -sh /home/user
uptime

5. 网络操作

以下指令用于网络相关操作：

  • ping: 测试网络连通性
  • ifconfig: 显示或配置网络接口
  • netstat: 显示网络连接状态
  • curl: 发送 HTTP 请求
  • wget: 下载文件
# 示例
ping www.google.com
ifconfig
netstat -an
curl http://example.com
wget http://example.com/file.zip

6. 压缩与解压

以下指令用于文件的压缩与解压：

  • tar: 打包文件
  • gzip: 压缩文件
  • unzip: 解压 ZIP 文件
# 示例
# 打包并压缩
 tar -czvf archive.tar.gz folder_name

# 解压
 tar -xzvf archive.tar.gz

# 解压 ZIP 文件
 unzip file.zip
